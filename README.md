# VolumeAnnotate
Virtual Unwrapping Tool for the Vesuvius Challenge

## Getting Started
### Installation

1. Clone the repo
```sh
git clone https://github.com/MosheLevy20/VolumeAnnotate.git
cd VolumeAnnotate
```
2. Install dependencies
```sh
pip install -r requirements.txt
```
### Running
To run simply do
```sh
python VolumeAnnotate.py
```
